# Attachments Plan

This folder is intended for evidence and supporting files that strengthen the proposal and complete the bid submission package.

## Recommended Contents

### Administrative and Compliance

- signed tender forms
- SBD documents
- POPIA consent form
- Integrity Pact
- SARS TCS evidence
- CSD report
- CIPC certificate
- SBD 6.1 and preference-point evidence

### Commercial

- pricing schedules
- commercial assumptions
- financial statements or management accounts

### Team Evidence

- project manager CV
- software developer CV
- solutions architect CV
- business analyst CV
- qualification copies

### Reference Evidence

- signed client reference letters
- letters of award

### Technical Evidence

- rendered architecture diagrams
- standards-based security plan
- risk assessment
- prototype screenshots or demo support material

### Branding

- bidder logo
- customer logo
- optional light / dark logo variants if required by export templates
