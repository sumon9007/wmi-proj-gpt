# Branding Assets

Place bidder and customer logo files for proposal export in this folder.

## Recommended Files

- `bidder-logo.png` or `bidder-logo.svg`
  - Waymark Infotech logo
- `customer-logo.png` or `customer-logo.svg`
  - Gauteng Provincial Treasury logo

## Preferred Formats

- `svg` preferred for Word and PDF export where available
- `png` acceptable if it has a clean transparent background
- Avoid low-resolution `jpg` files unless no better source exists

## Recommended Rules

- Use transparent-background files where possible
- Keep file names stable so export tooling and AI prompts can reference them consistently
- If you have multiple versions, keep only one primary export version with the exact names above

## Suggested Optional Variants

- `bidder-logo-dark.png`
- `bidder-logo-light.png`
- `customer-logo-dark.png`
- `customer-logo-light.png`

Use variants only if you know the export template needs light/dark alternatives.
