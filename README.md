# Consulting Delivery Platform

This workspace is a starter package for managing a customer project with Claude Co-Work.

## Purpose

Use this repository to keep all awarded-project artifacts in one structured place:

- RFP and submitted response
- customer-shared documents
- solution architecture and design
- implementation and delivery plans
- meeting notes and email records
- working Word, Excel, PowerPoint, and diagram files
- GitHub Pages / knowledge portal markdown
- Claude commands, prompts, rules, and templates

## Recommended flow

1. Put all source material into `01-input`
2. Update `PROJECT_CONTEXT.md` and `CUSTOMER_PROFILE.md`
3. Run the analysis and design commands from `08-ai/commands`
4. Review drafts in `05-deliverables`
5. Publish clean summaries into `06-knowledge-portal/docs`

## Core folders

- `01-input` - all source materials
- `02-analysis` - extracted insights and working analysis
- `03-solution` - solution and architecture content
- `04-delivery` - project and implementation management artifacts
- `05-deliverables` - client-ready drafts, reviews, finals, exports
- `06-knowledge-portal` - GitHub Pages / MkDocs content
- `07-working-files` - native Office and diagram files
- `08-ai` - Claude roles, commands, prompts, rules, templates

## Lifecycle

Use this document lifecycle where applicable:

`draft -> review -> final -> archive`
